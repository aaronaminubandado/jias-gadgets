# Jia’s Gadgets — Frontend

## Overview
Jia’s Gadgets Frontend is a React-based e-commerce interface designed to consume a REST API backend. It demonstrates storefront presentation, user flows, and readiness for real-world API integration.

The frontend is intentionally built as an iterative MVP, focusing on structure and extensibility rather than visual polish.

---

## Features
- Product storefront with demo data
- Page-based navigation
- Prepared API integration points
- Checkout flow designed to connect with Stripe
- Modular component structure

---

## Tech Stack
- React.js
- JavaScript
- REST API consumption
- Deployed on Netlify

---

## Backend Integration
The frontend is designed to integrate with the Jia’s Gadgets Backend API:
- Product listing
- Authentication
- Checkout and payments
- Order history

API integration is partially implemented and structured for easy extension.

---

## Deployment
- Frontend hosted on Netlify
- Environment variables used for API base URLs
